## 투자성과 페이지 (신규, 제목: "투자성과")

> trades + dividends 데이터로 계산 가능한 항목들

### 상단 요약 카드
- [ ] 총 실현손익 (전체 SELL 거래 기준)
- [ ] 총 평가손익 (현재가 - 평단가 × 수량)
- [ ] 총 수령 배당금
- [ ] 세 항목 합산 = 총 수익

### 차트
- [ ] 연도별 실현손익 bar chart
- [ ] 누적 실현손익 line chart (시간순)
- [ ] 배당 YoY 성장 chart

### 지수 비교
- [ ] 기준일 설정 UI (사용자 지정)
- [ ] 기준일 이후 내 실현수익률 계산
- [ ] 같은 기간 KOSPI / KOSDAQ / S&P500 / NASDAQ 등락률 (KIS API)
- [ ] 수익률 비교 표 or 차트

### 종목별 분석
- [ ] 종목별 실현손익 순위 테이블
- [ ] 최대 수익 거래 TOP 3 / 최대 손실 거래 TOP 3

### 거래 통계
- [ ] 승률 (수익 난 SELL ÷ 전체 SELL)
- [ ] 평균 수익률 (수익 거래), 평균 손실률 (손실 거래)
- [ ] 평균 보유 기간
